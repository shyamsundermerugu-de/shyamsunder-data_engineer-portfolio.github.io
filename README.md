# Shyamsunder Merugu Portfolio Website

A static, GitHub Pages-ready portfolio website for Shyamsunder Merugu.

## Files

- `index.html` - portfolio content
- `styles.css` - responsive UI styling
- `script.js` - mobile menu and project filters
- `assets/certifications/` - SVG certification badge placeholders
- `.nojekyll` - disables Jekyll processing for GitHub Pages

## Before publishing

1. Replace the LinkedIn placeholder URL in `index.html`.
2. Replace SVG certification placeholders with official badge images if you have them.
3. Decide whether you want to publish phone number publicly. This template intentionally uses email only.

## GitHub Pages deployment

### Option A: User site

1. Create a GitHub repository named exactly: `your-github-username.github.io`.
2. Upload all files from this folder to the repository root.
3. Commit changes to the `main` branch.
4. Open `https://your-github-username.github.io`.

### Option B: Project site

1. Create any public repository, for example: `portfolio`.
2. Upload all files to the repository root.
3. Go to repository **Settings** > **Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select branch `main` and folder `/root`.
6. Save.
7. Your site URL will usually be `https://your-github-username.github.io/portfolio/`.
